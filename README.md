# 六阿哥网iOS版

### 项目介绍

基于BaokanIOS改进、重写后台接口、重写优化app、还有那啥。。目前还在开发中。。。

### 项目结构

> LiuAGeIOS *项目主目录*
> ------------ Resource *资源目录*
> ------------ Classes *项目所有类文件*
> ------------------------ Application *应用级别类*
> ------------------------ Vendor *第三方库*
> ------------------------ Categories *分类*
> ------------------------ Model *公共模型*
> ------------------------ Utils *工具类*
> ------------------------ Module *模块目录*
> ------------------------------------------------ Main *主模块*
> ------------------------------------------------ News *新闻模块*
> ------------------------------------------------ Profile *个人中心模块*

### 开发环境

*XCode7.3* + *swift2.2* ，如果下载项目后，编译失败，请检查XCode版本是否满足。
