# 六阿哥网iOS版

## 项目介绍

资讯类客户端，数据来源我的个人网站 [六阿哥网](http://www.6ag.cn)。

## 开发环境

*XCode8.2.1* + *swift3.0* ，如果下载项目后，编译失败，请检查 `XCode` 版本是否满足。

## Android端

`Android` 端使用 `Android Studio2.2.3` 开发，功能基本和 `iOS` 端实现一致。

[Android端仓库](https://github.com/6ag/LiuAGeAndroid)
